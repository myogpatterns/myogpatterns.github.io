Sewing Skill Level â€“ Beginner

The Alpha Beanies are a fantastic starting point for anyone new to sewing performance gear. If you can sew a straight line and a zigzag, you can make these! The simple construction and low material investment make this a rewarding first project using technical fleece or other active knits.

  • Simple Seams: You'll be using few seams with zigzag stitches. Also a great first project for a new serger.
  • Hemming Knits: The Fitted Beanie teaches a simple, clean hem for knits with options for more advanced methods.
  • Lined Construction: The Slouch Beanie teaches a clean, enclosed-seam method for lining.

A domestic sewing machine is all you need. While a serger can speed up construction and give professional finish, it is optional. Patience and a new set of ballpoint or stretch needles are your most valuable tools.
